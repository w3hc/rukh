# Rukh

[Nest.js](https://nestjs.com/)-based AI agent starter kit.

## Install

```bash
pnpm i
```

## Run

```bash
# development
pnpm start

# watch mode
pnpm start:dev

# production mode
pnpm start:prod
```

## Test

```bash
# unit tests
pnpm test

# e2e tests
pnpm test:e2e

# test coverage
pnpm test:cov
```
